 The inclusion of 3D products on a website is a trend that not only impresses visually, but transforms the online shopping experience.


== Description ==
fuken Model Viewer is a plugin that allows you to insert and interact with 3d objects on your WordPress website. The adoption of this innovative technology in the presentation of products online is a strategic step for any business that seeks to stand out, offer differentiated value and increase customer satisfaction. These types of objects on the web will increase user engagement and increase your site's revenue.


== Features ==
1. No flash needed
2. Compatible with modern browsers such as Chrome, Firefox, Edge and Safari.
3. Augmented reality support
4. Supports both mouse and touch.
5. You can zoom in on the 3d object using touch.


== Instalation ==
1. Download the plugin, unzip it and move the unzipped folder to the "wp-content/plugins" directory in your WordPress installation.
2. In your admin panel, go to Plugins and you'll find FKN Model Viewer in the plugins section.
3. Enable the option to upload .glb files in your wordpress admin
4. Click on the 'Activate' button to use your new plugin right away.
5. Done


==How to use==
1. After installing and activating the fuken Model Viewer plugin on your WordPress website.
2. Upload a .glb file and click save
3. Copy the shortcode
4. Navigate to your blog post or page and use the editor to add a shortcode.
5. Click "Publish" to see your new web page with an interactive 3D object.
6. Congratulations, you are already enjoying your plugin
